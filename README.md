# CardioVision 

## Overview

CardioVision is a web-based application that evaluates an individual’s risk of heart disease using basic health and lifestyle information. It provides quick insights to promote early awareness of cardiovascular health risks.

---

## Features

* Multi-step health assessment form
* Calculates heart disease risk (Low / Moderate / High)
* Displays risk percentage with summary
* Provides personalized lifestyle recommendations
* Option to retake assessment

---

## How It Works

Users enter details such as:

* Age
* Blood Pressure
* Cholesterol Levels
* BMI
* Smoking Habits
* Diabetes Status
* Family History
* Physical Activity

The system processes this data and generates a risk score along with health suggestions.

## Modules
